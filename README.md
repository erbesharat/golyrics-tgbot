# golyrics-tgbot
A telegram bot for [golyrics](https://github.com/mamal72/golyrics)

## Installation
```bash
go get github.com/erbesharat/golyrics-tgbot
```

## Usage
```bash
TG_TOKEN="TELEGRAM-BOT-TOKEN" golyrics-tgbot
```
